# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-05-13

### Added
- Initial release of Snake and Ladder Game
- Support for 2-4 players
- 100-square game board
- 7 snakes and 9 ladders
- Dice rolling functionality (1-6)
- Player movement and collision detection
- Winner detection system
- Visual board display
- Interactive command-line gameplay
- Comprehensive unit tests (18 test cases)
- GitHub Actions CI/CD pipeline
- Code quality checks (flake8, pylint)
- Code coverage tracking
- MIT License
- Contributing guidelines
- Complete documentation

### Features
- ✅ Multi-player support (2-4 players)
- ✅ Random dice rolls
- ✅ Snake/Ladder collision detection
- ✅ Board visualization
- ✅ Player turn management
- ✅ Winner detection
- ✅ Game state tracking

### Testing
- ✅ 18 comprehensive unit tests
- ✅ Test coverage for all game mechanics
- ✅ Edge case testing
- ✅ Automated test execution

### Code Quality
- ✅ flake8 linting
- ✅ pylint analysis
- ✅ Code coverage reporting
- ✅ Python 3.8+ compatibility
